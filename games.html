My Minecraft Server

I started my Minecraft server in 2013 so I could do things my way.

Later, in 2015, I took a class on how to program in Java, and ever since then I have been creating plugins for my server.

The server website is https://www.shadowxcraft.net

ShadowKitPVP

ShadowKitPVP is a plugin that I created in partnership with another team member on my Minecraft server starting February of 2016. It is one of the early large projects that I worked on. It was developed over 360 commits. It primarily uses MySQL for player data storage.

This project has a lot of functionality. The features include:

  • Different battle groupings (Free-For-All, 1v1, 2v2, and Party)
  • Different battle types with different functionality, like building
  • The creation of parties, which are groups of players that battle together
  • Automated kit selection and customization menu
  • Kill-To-Death ratio stats
  • Game Spectating
  • A rank system with leaderboards

RollbackCore

This project is designed to allow a region to be backed up, then later restored. Backups are stored to a binary file of my own design. This is a public, open-source plugin.

View the resource page here: https://www.spigotmc.org/resources/rollbackcore.18517/

It is my most popular public resource with over 1600 downloads.

PvPBots

The PVP bot project is one of the larger ones I have worked on. The design goal was to create non-player characters whose behavior was like that of a real player. With the current version, the ability of a bot to navigate is realistic. I spawned a few bots into the KitPVP server to test it, without telling the actual players that the bots were fake. See this video to see the outcome. Since then, the bots have been improved.

I documented all of my progress! Click on this link for the playlist to view the videos.

ShadowSpirits

This is a project I am working on to add custom elements to the Prison server. The "Spirits" are moving blobs of particles with behavior. They spawn when players break blocks, and there is a tool used to capture spirits.

Hover your mouse over or tap the picture to animate it

ShadowPrison

This plugin is designed to automate the processes of the new Prison server. To do this, I integrated LuaJ, allowing me and my teammates to script scenes. It utilizes PVPBots with added functionality to guard the common areas.

This project has given me the chance to train my teammates how to script. For example, one teammate lives in England, and is far ahead of his classmates in Computer Science thanks to what I have taught him.

The picture is of a cannon that we automated. It uses math to calculate a cylinder for any angle we choose, allowing the player to change the angle of the large cannon.

ShadowParkour

ShadowParkour is a simple plugin that automates the parkour system on the server, and keeps track of times to create a leaderboard. The database is done with MySQL.

Oil-Lamps

This is a very simple project that I made public. It is available for download here: https://www.spigotmc.org/resources/oil-lamps.17842/

This project makes it so that fire can work as usual throught the world, while maintaining specific fire instances where fires are not supposed to be. It also supports user requested features, like making the fires harmless.
